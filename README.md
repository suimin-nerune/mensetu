# めんせつ★ノック (Mensetsu Knock) v1.0.0

どうも、YouTubeで20時から睡魔と闘うVtuberの彗眠ネルネじゃない方です🌜
受験生を応援したいと思いスマホで完結する、中高生・就活生向けの面接練習Webアプリを開発しました！
面倒な登録は不要。アプリのようにホーム画面に追加して、スキマ時間の3分で面接練習ができます。

## 🔒 安心・安全設計
このアプリは **完全なオフライン処理** で動作します。
- 録音したあなたの声
- 登録した「My質問」
- 練習の履歴

これらのデータはお使いのスマホ（ブラウザ）の中にのみ保存され、**外部のサーバーへ送信されることは一切ありません**。
学校での利用や、プライバシーを気にする方でも安心してお使いいただけます。

## 🌐 アプリを使う
こちらのURLからブラウザですぐに使えます（インストール不要）：
https://suimin-nerune.github.io/mensetu/

## ✨ 主な機能
- **音声読み上げ (TTS)**: 面接官役（CV: 機械音声）が質問を読み上げます。
- **話す速さ調整**: ゆっくり話してほしい時や、早口の練習に対応。
- **録音・再生**: 自分の声を録音して、客観的にチェック。
- **3つのコース**: 
  - 🏫 高校受験
  - 🎓 大学・専門学校
  - 💼 就職・公務員
- **電車モード**: 声を出せない場所でも、文字だけでシミュレーション練習が可能。
- **My質問機能**: 自分だけのオリジナル質問を追加して練習できます。
